# Grid Protocol

> A self-evolving skill system for human-AI collaboration.

🌐 [grid-protocol.org](https://grid-protocol.org)

---

## ⚠️ Do NOT clone this repository

This is not a library or application to install. It is a seed — fetched by an AI model to bootstrap the Grid into your project.

---

## Initialize the Grid

Tell your AI model:

> *"Initialize the Grid from G667 at GitHub"*

The model will read `SKILL.md` from this repository and follow the initialization steps collaboratively with you. No cloning. No installer. No scripts to run manually — the model handles everything, one confirmed step at a time.

---

## What is the Grid?

The Grid is a self-initializing, self-evolving skill system that gives AI models persistent memory and structured collaboration context — session by session, project by project.

It is not a plugin. It is not a configuration file. It is a living foundation for human-AI collaboration.

---

## Compatibility

All skill files follow the [Agent Skills open standard](https://agentskills.io/specification). Skills work with any compatible agent — with or without the Grid.

Supported MCPs: GitHub Copilot CLI, Claude Code, Cursor, Windsurf.
